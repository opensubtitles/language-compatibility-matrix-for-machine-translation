Language Compatibility Matrix

Overview

This project provides comprehensive language compatibility scores for machine translation systems, featuring paired proximity scores, optimal source languages, and multi-model analysis. Perfect for researchers, developers, and ML engineers working on multilingual NLP applications.

85 Languages
7,225 Language Pairs

Key Features

Paired Proximity Scores

Asymmetric compatibility scores showing how well languages translate between each other

Optimal Source Languages

Identifies the best intermediate languages for multi-step translation chains

Multi-Model Analysis

Supports Google Translate, DeepL, and perplexity-based evaluation

Quick Start

Python Example

from language_compatibility import LanguageCompatibilityMatrix # Initialize with default dataset lcm = LanguageCompatibilityMatrix() # Find best source language for translating to Japanese best_sources = lcm.get_best_sources('ja', top_k=5) print(f"Best sources for Japanese: {best_sources}") # Get compatibility score between two languages score = lcm.get_compatibility_score('en', 'ja') print(f"English→Japanese compatibility: {score}")

JavaScript Example

import { LanguageCompatibilityMatrix } from 'language-compatibility-matrix'; const lcm = new LanguageCompatibilityMatrix(); // Find optimal translation chain const chain = lcm.findOptimalChain('en', 'ja', ['de', 'fr']); console.log('Optimal translation chain:', chain); // Check bidirectional compatibility const bidirScore = lcm.getBidirectionalScore('en', 'ja'); console.log('Bidirectional compatibility:', bidirScore);
