Manus v1 Dataset Documentation

Manual Curation and Refinement of Language Translation Compatibility Scores

Dataset Overview:
  • Name: Manus v1 Language Compatibility Dataset
  • Languages: 57 languages (ISO 639-1 codes only)
  • Total Pairs: 3,192 directional language pairs
  • Scale: 0-255 compatibility scores
  • Approach: Manual curation and expert refinement

üìã Methodology

Data Collection

The Manus v1 dataset represents a manually curated collection of language translation compatibility scores, developed through systematic expert analysis and refinement.

Scoring System

All scores are on a 0-255 scale where:

Score Range Compatibility Level Description
250-255 Near-Perfect Extremely high translation quality, minimal information loss
200-249 Excellent Very high quality translations, suitable for professional use
150-199 Good Good translation quality with acceptable accuracy
100-149 Moderate Moderate quality, may require post-editing
50-99 Limited Limited compatibility, significant challenges
0-49 Low Poor translation quality, major structural differences

üåç Language Coverage

The dataset includes 57 languages using ISO 639-1 two-letter codes, covering major world languages across multiple language families:

  • Indo-European: Romance, Germanic, Slavic, Indo-Aryan branches
  • Afro-Asiatic: Semitic languages
  • Sino-Tibetan: Chinese variants
  • Japonic: Japanese
  • Koreanic: Korean
  • Turkic: Turkish and related languages
  • Uralic: Finnish, Hungarian
  • And more...

üìä Dataset Characteristics

Key Features:
  • Focus on two-letter ISO 639-1 codes for better compatibility
  • Manual expert curation for accuracy
  • Balanced coverage across language families
  • 3,192 directional language pairs
  • Optimized for practical translation workflows

JavaScript Example

// Load dataset
const response = await fetch('datasets/language-pairs-translation-proximity-manus-v1.json');
const manusData = await response.json();

// Get compatibility score
const score = manusData['en']['es']; // English to Spanish
console.log(`English ‚Üí Spanish: ${score}/255`);

Python Example

import json

# Load dataset
with open('datasets/language-pairs-translation-proximity-manus-v1.json', 'r') as f:
    manus_data = json.load(f)

# Get compatibility score
score = manus_data['en']['es']
print(f"English ‚Üí Spanish: {score}/255")
