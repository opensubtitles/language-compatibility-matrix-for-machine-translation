Scoring Methodology

The dataset uses a 0-255 scale to represent translation compatibility between language pairs. Higher scores indicate better translation quality and easier cross-lingual transfer.

Score Interpretation:

  • 250-255: Near-perfect compatibility (e.g., Czech‚ÜîSlovak, Norwegian‚ÜîDanish)
  • 200-249: Excellent compatibility (close language pairs within families)
  • 150-199: Good compatibility (related languages with shared features)
  • 100-149: Moderate compatibility (distant relatives or some structural similarity)
  • 50-99: Limited compatibility (different families, some transfer possible)
  • 0-49: Low compatibility (very distant languages, minimal transfer)

Special Considerations

Dialectal Variations

  • Norwegian (Bokm√•l vs. Nynorsk): Scores represent Bokm√•l as default
  • Chinese (Mandarin vs. Cantonese): Scores based on Standard Mandarin
  • Arabic (MSA vs. Dialects): Modern Standard Arabic used as baseline
  • Portuguese (European vs. Brazilian): Merged as single language variant

Script System Impact

Script Compatibility Penalties:

  • Same script family (Latin-Latin, Cyrillic-Cyrillic): No penalty
  • Different alphabetic scripts (Latin-Cyrillic): -5 to -10 points
  • Alphabetic to logographic (Latin-Chinese): -15 to -25 points
  • Alphabetic to abugida (Latin-Devanagari): -10 to -15 points
