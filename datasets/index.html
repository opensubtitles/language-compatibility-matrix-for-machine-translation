Language Translation Proximity Datasets

JSON datasets for machine translation compatibility analysis

What are these datasets?

These JSON files contain language pair compatibility scores (0-255 scale) that indicate how well languages translate between each other. Higher scores mean better translation quality and easier cross-lingual transfer.
